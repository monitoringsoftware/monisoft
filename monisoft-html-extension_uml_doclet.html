MoniSoft API documentation
This documentation describes the source of the MoniSoft monitoring application.
 
The best starting points and the most important classes are in the base package (main application and main-GUI) and the components package.
As central elements, Sensors are represented by SensorProperties and buildings by BuildingProperties.
The calculation of measurement intervals (interpolation and aggregation) is handeled by classes in the de.jmonitoring.DataHandling.Interpolators package.
